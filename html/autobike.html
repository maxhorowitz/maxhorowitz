AutoBike

Mission

To build a mechanism that automatically shifts the gears on my bike using an Arduino microcontroller and various sensors and motors.

Technology

Arduino is an open source electronics platform based on easy-to-use hardware and software. It is intended for anyone making interactive projects. I used Arduino to build an automatic-transmission bike. Essentially, just like an automatic-transmission car, this bike shifts gears on it own based on RPM readings.

The Arduino microcontroller uses a magnetic reed switch and its built-in timer to measure the elapsed time between each full pedal rotation. Depending on the Arduino's calculation of the user's rotations per minute, it uses a series of servo motors to upshift or downshift gears if necessary. A Liquid Crystal Display attached to the handlebars of the bike shows the current gear and RPM readings, each of which is updated at 60 Hz.

Credits

This project could not have been completed without help from Amanda Ghassaei's Arduino Bike Speedometer project, which she has graciously posted on Instructables.com. I modified her program to function as an RPM calculator, which then uses its output readings to determine whether the bike should change gears. Her project is published at the following link: https://www.instructables.com/id/Arduino-Bike-Speedometer/

Elements

Arduino Mega 2560 R3 Microcontroller, Magnetic Reed Switch, LCD Display, HV2060MG Digital Servo Motor, DS3120MG Digital Servo Motor, 6 Volt 12 AH Sealed Lead Acid Battery
